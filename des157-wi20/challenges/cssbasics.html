Inheritance in CSS

Inheritence is the property in code where certain objects/values can adopt specific properties based on the property of its parent. For example, an inherited property is "color", for where body { color: #000000 }, every tag in the body will have that color due to having inherited from the body tag.

Specificity in CSS

When there are two or more conflicting CSS rules about a certain tag, the rule that is more specific based on inline styles, IDs, classes/attributes, and elements (such as h1) is applied. A compound selector in CSS is a selector that can use different types to focus in on one element (ex: #some-id .some-class div {}), whereas multiple selector allows you to apply the same style to multiple elements (ex: div#main, div#nav {}).

ID and Class Attributes

An ID can be used to identify one element and must be unique within the HTML code; it uses "#" (ex: #text1 {}) while a class can be used to identify more than one element and uses "." (ex: .text1 {}).
