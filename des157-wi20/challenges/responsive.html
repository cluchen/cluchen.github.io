About Media Queries

Media queries are @media rules in CSS that allow for responsive design - you can create CSS specifications for different window sizes, such as 600px. For min-width media quaries, the code following only applies for widths greater than or equal to x width; for max-width media queries, the code following only applies for widths smaller than or equal to x width. We design from small to big screens, since smaller screens are harder to design for - there's less window space to fill in content, and elements often need to be modified or relayed on top of each other.

Relative Units

Relative units are units that are relative to everything else, and it's possible to make it so that the size of the unit is relative to the other elements on the page. The em unit makes it relative to the size of the parent element, and vw is 1% of the viewport's width while vh is 1% of the viewport's height.

Meta Viewport Tag

This meta viewport tag sets the width and height of the viewport to the width and height of the device. The page is not responsive and the meta tags aren't of use without the meta viewport tag.
