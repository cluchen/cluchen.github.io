Why we use the Normalize CSS Script

Normalize CSS is a library used for normalizing the styling between HTML elements, as a sort of "default" style. It provides consistency to HTML files across browsers. For example, the h2 tag can appear to be similar in style / size / etc. across other browers no matter the viewing platform. Thus, it's a common library used.

What is Semantic Markup

Semantic Markup are tools in HTML such as tags (h1, h2, a, etc). These tags provide meaning to the code / text within -- for example, an a tag has the semantic meaning of being a link. Thus, semantic markup allows the the code to have properly defined constraints and meaning, and that HTML elements are being used for their defined purpose.
